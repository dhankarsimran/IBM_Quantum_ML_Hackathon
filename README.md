# IBM Quantum ML Hackathon ⚛️

## Quantum vs Classical Machine Learning on CAR T-Cell Data

This project was developed during the IBM Quantum ML Hackathon to compare Classical Machine Learning models with Quantum-Enhanced Machine Learning approaches on the CAR T-Cell biomedical dataset.

We explored how quantum feature encoding and hybrid quantum-classical workflows can improve learning performance compared to traditional ML techniques. The project focused on implementing a Quantum SVM pipeline using Qiskit and benchmarking it against classical SVM models.

---

## Project Theme

**Quantum-Enhanced Machine Learning for Biomedical Data Analysis**

---

## Key Highlights

* Built and compared Classical SVM and Quantum SVM models
* Applied quantum feature encoding using Qiskit
* Used hybrid quantum-classical learning workflows
* Achieved ~7% improvement in F1-score using Quantum SVM over the classical baseline
* Visualized performance metrics and learning behavior across models
